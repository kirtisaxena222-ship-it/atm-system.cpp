# 🏧 ATM System (C++)

A simple console-based ATM system built using C++.

## Features
- PIN Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- Menu Driven System

## Technologies Used
- C++
- OOP Concepts

## How to Run
1. Compile using:
   g++ main.cpp -o atm
2. Run:
   ./atm

Default PIN: 1234
